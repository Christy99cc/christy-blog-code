> 需要准备：

> Linux系统 （本文使用的是Ubuntu 18.04）
> 
> 一个拥有sudo权限的账号user


1. 创建账号

```
 sudo adduser zsx
```

同时会创建用户主目录`/home/zsx/`

2. 为用户增加sudo权限

```
sudo vim /etc/sudoers 
```

增加权限

```
# User privilege specification
root    ALL=(ALL:ALL) ALL
zsx ALL=(ALL:ALL) ALL
```
